# Scripts

Utility scripts for various tasks.

## Bash

### Helper scripts

- `/bash/helper_scripts/colors.sh` - Script containing ANSI color definitions for terminal output.

### Main scripts

- `/bash/llama-cpp_curl_openai-api_with_streaming.sh` - Sends a prompt to a local OpenAI compatible API (e.g., `llama.cpp`) using `curl`. It handles streaming response, echoing it to stdout and writing output to a file.
